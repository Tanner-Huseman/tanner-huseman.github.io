# tanner-huseman.github.io

Personal portfolio — data platform engineer, former Broadway dancer. Built with Astro, GSAP, and Three.js.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🏗️ Tech Stack

- **Framework**: [Astro](https://astro.build/) with React islands
- **Animations**: [GSAP](https://greensock.com/gsap/) + ScrollTrigger
- **3D**: [Three.js](https://threejs.org/) / React Three Fiber
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Content**: MDX
- **Hosting**: GitHub Pages
- **CI/CD**: GitHub Actions

## 📁 Project Structure

```
src/
├── layouts/        # Base HTML layout
├── pages/          # Routes (index + project detail pages)
├── components/     # Astro + React components
├── content/        # MDX files (projects, articles)
├── styles/         # Global CSS + design tokens
├── lib/            # GSAP + Three.js utilities
└── assets/         # Images, logos, fonts
```
